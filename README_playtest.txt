HOLLOW — Early Access v0.1
==========================

A first-person horror voxel game. No Java install required — everything is bundled.

Run Hollow.exe to start.

---------------------------------------------------------------------------
CONTROLS
---------------------------------------------------------------------------
WASD              Move
Mouse             Look
Space             Jump
Shift             Sprint
Left Ctrl         Sneak (quiet movement)
1–9               Select hotbar slot
Left click        Mine block / attack
Right click       Place block / interact / eat
I                 Open / close inventory
E                 Interact (pick up items, open/close doors)
F2                Save world manually
F3                Debug overlay (coordinates, FPS)
F11               Toggle fullscreen
R                 Recipe book
ESC               Pause menu / close menus

---------------------------------------------------------------------------
OBJECTIVE
---------------------------------------------------------------------------
Find all 5 RELICS hidden across the world — one in each major area.

  - Two are somewhere in the forest and hostile wastes (surface)
  - One is deep underground
  - Two require following clues into the dead zones

Craft a COMPASS (Crystal + Wood Plank) to track the nearest relic.

Once you have all 5, return to the ORIGIN (world centre, marked by the
purple beacon column). Step into the Void Gate to escape.

---------------------------------------------------------------------------
THE LIMINAL ZONES
---------------------------------------------------------------------------
Some portals lead to spaces outside the normal world.

  ZONE 1 — THE MEADOW
    A strange flat plain with an old building at its centre.
    A portal inside the building leads deeper.

  ZONE 2 — HOLLOWAY MANOR
    A sealed two-storey mansion. There is no way out.
    Something walks the halls. Survive long enough to understand why.

---------------------------------------------------------------------------
CRAFTING
---------------------------------------------------------------------------
Press I to open your inventory.  Arrange items in the 2x2 grid to craft.
Press R at any time to view all recipes.

Some recipes require a Workbench (3x3 grid). Place one and stand near it
while the crafting panel is open.

Useful early recipes:
  Wood Log (2x2)        →  Wood Plank x4
  Wood Plank (2x2)      →  Workbench
  Coal + Wood Plank     →  Torch x8
  Torch + Crystal       →  Lantern
  Crystal + Wood Plank  →  Compass

---------------------------------------------------------------------------
TIPS
---------------------------------------------------------------------------
- Equip armor (Bone / Stone / Crystal) by right-clicking it from hotbar
- Standing on FUNGUS blocks slowly heals you
- Breaking FUNGUS drops Cave Mushrooms (+10 HP when eaten)
- Eat Raw Pork or Cooked Pork for health recovery
- Keep a torch or lantern nearby underground — darkness makes things worse
- Listen carefully. Some threats announce themselves. Others do not.
- Fall damage triggers — long drops hurt. Very long drops kill.

---------------------------------------------------------------------------
SAVE FILES
---------------------------------------------------------------------------
Saves are stored in the "worlds" folder next to Hollow.exe.
The game auto-saves every 5 minutes. Manual save: F2.

---------------------------------------------------------------------------
KNOWN ISSUES / EARLY ACCESS NOTES
---------------------------------------------------------------------------
- This is an early access build. Expect rough edges and ongoing changes.
- Gamepad / controller not yet supported.
- Some procedural events are louder than intended on certain hardware.
- Zone 2 (Holloway Manor) has no exit by design — death respawns you.
